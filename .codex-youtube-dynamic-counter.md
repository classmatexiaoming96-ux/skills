# Task Brief — 将 youtube_course 课程计数器改为动态统计

## 目标

把 `classmatexiaoming96-ux.github.io` 静态站 `youtube_course/index.html` 的课程/时长/分类计数器从硬编码改为 JS 动态统计，让后续加新视频子页时无需手动改数值。

## 当前状态

index.html 第 399-415 行有 3 个硬编码 stat：

```html
<div class="stats-bar">
  <div class="stat-item">
    <div class="stat-value">11</div>        ← 硬编码，实际有 12 个页面
    <div class="stat-label">精选课程</div>
  </div>
  <div class="stat-item">
    <div class="stat-value">~320</div>      ← 硬编码
    <div class="stat-label">分钟时长</div>
  </div>
  <div class="stat-item">
    <div class="stat-value">3</div>          ← 硬编码（分类数）
    <div class="stat-label">内容分类</div>
  </div>
</div>
```

## 需要改动的文件

`/root/repos/classmatexiaoming96-ux.github.io/youtube_course/index.html`

## 实现方案

在 `</body>` 前插入一段内联 `<script>`。因为这是纯前端 GitHub Pages 站，没有构建步骤。

### 统计逻辑

1. **精选课程数**：统计 `<main>` 内所有 `href="*.html"` 的链接数（去重，排除 `index.html` 自身）
2. **分钟时长**：统计所有 `class="duration"` 或卡片区形如 `⏱ N 分钟 / ~N 分钟 / N 秒` 的文本，统一换算为分钟取整（≤1 分钟按 1 算），加总后显示 `~{total}`。**如果提取太复杂，这个可以跳过保持硬编码，只必有课程数自动统计。**
3. **内容分类数**：统计 `section` 或分类标题（`h2`）数量

### 具体 selector 建议

看实际 HTML 结构（文件在 `/root/repos/classmatexiaoming96-ux.github.io/youtube_course/index.html`）：

```javascript
// 课程数：每个 video-card 的 href
const links = document.querySelectorAll('.video-card a[href$=".html"]');
// 去重
const courses = [...new Set([...links].map(a => a.getAttribute('href')))].filter(h => h !== 'index.html');

// 分类数：二级标题
const categories = document.querySelectorAll('.category-section h2, section > h2');

// 更新 stat-value 们
document.querySelectorAll('.stat-item')[0].querySelector('.stat-value').textContent = courses.length;
document.querySelectorAll('.stat-item')[2].querySelector('.stat-value').textContent = categories.length;
```
